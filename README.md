TaskFlow – Employee Task Management System

A modern, interactive, and AI-powered employee task management web application built with React (via CDN), TailwindCSS, and LocalStorage.
Designed for hackathons — fast to build, visually stunning, and feature-rich.

Features
1. Secure Login System

Demo user accounts (Admin, Manager, Employee)

Local authentication using sample credentials

2. Modern Dashboard

Real-time task stats: Pending, In-Progress, Completed

Animated cards & fade-in UI for smooth UX

3. Task Management System

Create, edit, delete tasks

Assign to employees

Set priority (High, Medium, Low)

Change task status

Add due dates

Auto-color-coded task cards

4. Smart Deadline Reminder

Shows tasks due in the next 7 days

Highlights urgent tasks

Hide/show reminder

5. AI Chat Assistant 

Helps users navigate UI

Can highlight specific sections

Quick-action buttons

Learns from message intent

Examples:

“Show my tasks”

“Where is search?”

“Show team members”

6. Employee Directory

View all employees

Click on an employee → See all their tasks

Sorting by:

Total tasks

Completed tasks

Pending tasks

Overdue tasks

Department

Performance

And more!

7. Fully Responsive UI

TailwindCSS design

Works on desktop, tablet, and mobile

8. Zero Backend Needed

Uses LocalStorage for data

Perfect for demos and hackathons

Tech Stack
Frontend

HTML

TailwindCSS

React (CDN – no build tools required)

State & Data

LocalStorage
